## ğŸ“– Overview

The Titanic shipwreck is one of the most famous disasters in history. In
this project, we build Machine Learning models to predict passenger
survival based on features like age, gender, ticket class, fare, and
more.

The project includes:

-   Data preprocessing
-   Missing value handling
-   Feature engineering
-   Exploratory Data Analysis (EDA)
-   Model training and comparison
-   Final prediction generation

### Models Used

-   Logistic Regression
-   Decision Tree
-   Random Forest
-   Extra Trees Classifier
-   XGBoost

**Best Model:** Random Forest
**Accuracy Achieved:** ~82% -- 85%
